InvoiceTable
============

Объект предназначен для редактирования табличной части счета-фактуры.


Свойства объекта
----------------

- **TotalWithVatExcluded** (число с плавующей запятой, чтение/запись) - сумма без учета НДС

- **Vat** (число с плавующей запятой, чтение/запись, обязательно для заполнения) - сумма НДС

- **Total** (число с плавующей запятой, чтение/запись, обязательно для заполнения) - сумма всего

- **TotalNet** (число с плавующей запятой, чтение/запись) - нетто всего

- **Items** (:doc:`коллекция <Collection>` объектов :doc:`ExtendedInvoiceItem <ExtendedInvoiceItem>`, чтение) - информация о товарах


Методы объекта
--------------

-  :doc:`AddItem <AddItem-(InvoiceTable)>` - добавляет строку в табличную часть счета-фактуры
