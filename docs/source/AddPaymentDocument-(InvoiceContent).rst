AddPaymentDocument 
===================================

Метод объекта :doc:`InvoiceContent <InvoiceContent>`

**Синтаксис**


AddPaymentDocument()

**Возвращаемое значение**


Объект :doc:`PaymentDocument <PaymentDocument>`.

**Описание**


Добавляет сведения в коллекцию платежно-расчетных документов.
