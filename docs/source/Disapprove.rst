Disapprove
==========

Метод объекта :doc:`Document <Document>`

**Синтаксис**


Disapprove(<Comment>)

**Параметры**


-  <Comment> (строка, необязательный) - комментарий к отказу от
   согласования

**Возвращаемое значение**


Отсутствует

**Описание**


Ставит признак отказа в согласовании документа.
