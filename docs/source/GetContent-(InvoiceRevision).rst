GetContent 
============================

Метод объекта :doc:`InvoiceRevision <InvoiceRevision>`.

**Синтаксис**


GetContent()

**Возвращаемое значение**


Объектная модель данных :doc:`InvoiceContent <InvoiceContent>`.

**Описание**


Возвращает содержание исправленного счет-фактуры в виде объектной
модели.
