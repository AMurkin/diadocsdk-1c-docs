COM Diadoc API
==============

COM Diadoc API - внешняя компонента, предназначенная для интеграции с учётными системами посредством COM-интерфейса, в частности - с платформой 1С.

Данный способ позволяет максимально быстро решать типовые задачи, возникающие при интеграции с Диадоком.
Компонента может запускаться на машинах с ОС Microsoft Windows не требуя установки каких-либо дополнительных модулей.
