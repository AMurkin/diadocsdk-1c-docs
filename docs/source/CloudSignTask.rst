CloudSignTask
=============

Объект предназначен для подписания набора документов облачной подписью.

Свойства
--------

У объекта нет доступных свойств.


Методы
------

-  :doc:`AddContent <AddContent>` - добавляет содержимое документа
   в список заданий на подписание облачной подписью
-  :doc:`Sign <Sign>` - передает на сервер запрос на подписание документов,
   добавленных в список и инициирует отправку СМС с кодом подтверждения на телефон пользователя
-  :doc:`Confirm <Confirm>` - подтвержает использование облачной подписи для
   подписания набора документов с помощью введенного пользователем кода подтверждения.
