CloudSignTask
=============

.. deprecated:: 5.26.0
    На данный момент вопрос поддержки работы с Контур.Сертификатами не решён

Объект предназначен для подписания набора документов электронной подписью СКБ Контур.

Свойства
--------

У объекта нет доступных свойств.


Методы
------

-  :doc:`AddContent <AddContent>` - добавляет содержимое документа в список заданий на подписание
-  :doc:`Sign <Sign>` - передает на сервер запрос на подписание документов, добавленных в список и инициирует отправку СМС с кодом подтверждения на телефон пользователя
-  :doc:`Confirm <Confirm>` - подтвержает использование электронной подписи СКБ Контур для подписания набора документов с помощью введенного пользователем кода подтверждения.

Пример использования см. в разделе :doc:`Как работать с электронной подписью СКБ Контур <How-use-cloud-sign>`.
