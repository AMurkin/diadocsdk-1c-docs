TestConnection
==============

Метод :doc:`объекта интерфейса "Диадок" <Объект-интерфейса-Диадок>`.

**Синтаксис**


TestConnection()

**Возвращаемое значение**


Булево.

**Описание**


Происходит попытка соединения с сервером Диадок:

В случае успешного соединения, возвращается Истина, в противном случае,
возвращается Ложь.
