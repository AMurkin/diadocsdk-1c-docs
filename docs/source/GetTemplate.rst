GetTemplate
===========

Метод объекта :doc:`Organization <Organization>`


**Синтаксис**

GetTemplate(<Id>)


**Параметры**

-  <Id> (строка, обязательный) - идентификатор шаблона


**Возвращаемое значение**

:doc:`Template <Template>`


**Описание**

Возвращает шаблон по его идентификатору
