GetRejectionComment 
==============================================

Метод объекта :doc:`XmlAcceptanceCertificate <XmlAcceptanceCertificate>`.

GetRejectionComment()

Строка. Комментарий к отказу в подписании.

**Описание**


Возвращает комментарий к отказу в подписании.
