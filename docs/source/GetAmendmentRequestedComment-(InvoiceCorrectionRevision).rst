GetAmendmentRequestedComment 
========================================================

Метод объекта :doc:`InvoiceCorrectionRevision <InvoiceCorrectionRevision>`.

GetAmendmentRequestedComment()

Строка. Комментарий к уведомлению об уточнении.

**Описание**


Возвращает комментарий к уведомлению об уточнении.
