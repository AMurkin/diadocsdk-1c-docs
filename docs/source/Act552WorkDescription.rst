Act552WorkDescription
=====================

Описание выполненных работ для документа типа "Акт в
формате 552-го приказа ФНС".

Свойства
--------

- **StartingDate** (дата, чтение/запись) - начало периода выполнения работ

- **CompletionDate** (дата, чтение/запись) - окончание периода выполнения работ

- **TotalWithVatExcluded** (число c плавающей точкой, чтение/запись) - сумма без учета НДС - итого

- **TotalVat** (число c плавающей точкой, чтение/запись) - сумма НДС - итого

- **Total** (число c плавающей точкой, чтение/запись) - сумма с учетом НДС - итого

- **Items** (коллекция :doc:`Collection <Collection>` объектов :doc:`Act552WorkItem <Act552WorkItem>`) - сведения о произведенной работе

Методы
------

-  :doc:`AddItem <AddItem-(Act552WorkDescription)>` - добавляет элемент в коллекцию сведений о произведенной работе
