Как работать с документами
==========================

Все действия с документами производятся в контексте объекта :doc:`Organization <Organization>`

:doc:`Документы <Document>` в Диадоке имеют разные типы. Существуют свойства и методы, характерные как для всех типов, так и специфичные для какого-то конкретного типа.
Для получения информации о типах доступных документов используйте :func:`Organization.GetDocumentTypes`.
