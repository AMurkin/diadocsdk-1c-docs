GetDocumentPackage
==================

Метод объекта :doc:`Document <Document>`.

**Синтаксис**


GetDocumentPackage()


**Возвращаемое значение**


:doc:`DocumentPackage <DocumentPackage>`.


**Описание**


Если документ, для которого вызывается этот метод, является частью пакета,
то этот метод вернет такой пакет.
