TovTorgTable
============

Сведения таблицы Торг-12 - сведения об ассортименте, количестве, стоимости и другой информации о товарных позициях

Свойства объекта
----------------


- **Items** (коллекция :doc:`Collection <Collection>` объектов типа :doc:`TovTorgItem <TovTorgItem>`) - информация о товарах

- **TotalQuantity** (число с плавающей точкой, чтение/запись) - количество мест всего по документу

- **TotalGross** (число с плавающей точкой, чтение/запись) - масса брутто всего по документу

- **TotalNet** (число с плавающей точкой, чтение/запись) - масса нетто всего по документу

- **TotalWithVatExcluded** (число с плавающей точкой, чтение/запись) - сумма без учета налога

- **TotalVat** (число с плавающей точкой, чтение/запись) - сумма НДС - всего по документу

- **Total** (число с плавающей точкой, чтение/запись) - сумма всего

--------------
