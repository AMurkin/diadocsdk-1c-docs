LoadBuyerTitleFromFile
=======================

Метод объекта :doc:`TemplateToSend <TemplateToSend>`


**Синтаксис**

LoadBuyerTitleFromFile(<Path>)


**Параметры**

-  <Path> (строка, обязательный) - путь к файлу с шаблоном документа


**Возвращаемое значение**

Отсутствует


**Описание**
Задаёт содержание титула покупателя в шаблоне документа на отправку
