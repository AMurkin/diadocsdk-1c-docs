GetFileContent
==============

Метод объекта :doc:`DocumentEvent <DocumentEvent>`.

**Синтаксис**


GetFileContent(<Path>)

**Параметры**


-  <Path> (строка, обязательный) - путь для сохранения содержимого
   документа

**Возвращаемое значение**


Строка. Исходное имя файла документа.

**Описание**


Сохраняет содержимое документа на диск.
