SetOneSDocumentId
=================

Метод объекта :doc:`Document <Документы>`

**Синтаксис**


SetOneSDocumentId(<OneSDocumentId>)

**Параметры**


-  <OneSDocumentId> (строка, обязательный) - идентификатор документа в
   1С

**Возвращаемое значение**


Отсутствует

**Описание**


Устанавливает идентификатор 1С для документа.
