Send
====

Метод объекта Send

**Синтаксис**


Send()

Объект, производный от :doc:`Document <Document>`


**Описание**

Производит отправку сообщения на сервер Диадок.


Метод больше не существует. Используйте :doc:`Send <Send-(PackageSendTask)>`
