5.14.0
------

.. feed-entry::
  :date: 2017-01-20



.. rubric:: Добавлено

* Поддержка отправки *Торг-12*, *Акта* и *Счёта фактуры* в формате УПД:

  * Добавлены новые значения параметров для методов :func:`Organization.CreateSendTask`, :func:`Organization.CreateSendTaskFromFile`, :func:`PackageSendTask.AddDocument`, :func:`PackageSendTask.AddDocumentFromFile`

* Добавлен метод :func:`Organization.CreateSendTaskFromFileRaw`
* Добавлен метод :func:`PackageSendTask.AddDocumentFromFileRaw`
* Добавлен реквизит **IsEncryptedContent** для объекта :doc:`../../ComObjects/Document`

* Добавлена возможность отправки зашифрованных документов *Акт*, *Торг-12*, *Счёт-Фактура*:

  * Добавлен метод :func:`Counteragent.GetCertificates`
  * Добавлен метод :func:`SendTask.AddEncryptCertificate`
  * Добавлен метод :func:`PackageSendTask.AddEncryptCertificate`



.. rubric:: Изменено

* Контент объектов :doc:`../../ComObjects/Invoice`, :doc:`../../ComObjects/XmlTorg12`, :doc:`../../ComObjects/XmlAcceptanceCertificate` теперь может быть :doc:`../../ComObjects/UtdSellerContent` и :doc:`../../ComObjects/UtdBuyerContent`
* Метод **AddCertToFnsRegistrationMessage** переименован в :func:`Organization.SendFnsRegistrationMessage`



.. rubric:: Исправлено

* Метод :func:`Document.SaveAllContent` теперь работает и для *УПД*
* Метод :func:`Document.SaveAllContentAsync` теперь работает и для *УПД*
* При вызове :func:`Document.MarkAsRead` документ не помечался какпрочитанный
