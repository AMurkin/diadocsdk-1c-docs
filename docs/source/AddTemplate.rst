AddTemplate
===========

Метод объекта :doc:`TemplateSendTask <TemplateSendTask>`


**Синтаксис**

AddTemplate(<Type>)


**Параметры**

Возможные значения параметра Type можно получить из поля Name :doc:`описаний типов документов <DocumentTypeDescription>`, полученных методом :doc:`Organization.GetDocumentTypes() <GetDocumentTypes>`


**Возвращаемое значение**

Объект шаблона на отправку :doc:`TemplateToSend <TemplateToSend>`


**Описание**

Добавляет шаблон заданного типа в :doc:`задание на отправку <TemplateSendTask>`
