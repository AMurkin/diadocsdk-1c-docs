Reject 
======================

Метод объекта :doc:`Nonformalized <Nonformalized>`.

**Синтаксис**


Reject(<Comment>)

**Параметры**


-  <Comment> (строка, обязательный) - комментарий к отказу в подписании.

**Возвращаемое значение**


Отсутствует

**Описание**


Формирует отказ в подписании документа.
