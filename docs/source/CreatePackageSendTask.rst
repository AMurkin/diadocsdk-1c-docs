CreatePackageSendTask
=====================

Метод объекта :doc:`Organization <Organization>`

CreatePackageSendTask()

Объект :doc:`PackageSendTask <PackageSendTask>`.

Позволяет создать задание для отправки пакета документов на
сервер Диадок.
