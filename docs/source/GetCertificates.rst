GetCertificates
===============

Метод объекта :doc:`Counteragent <Counteragent>`

**Синтаксис**


GetCertificates()

**Возвращаемое значение**


Коллекция :doc:`Collection <Collection>` объектов :doc:`PersonalCertificate <PersonalCertificate>`.

**Описание**


Возвращает список сертификатов для контрагента.
