AddSigner
=========

Метод объекта :doc:`Act552SellerContent <Act552SellerContent>`


**Синтаксис**

AddSigner()


**Возвращаемое значение**

Объект :doc:`ExtendedSigner <ExtendedSigner>`


**Описание**

Добавляет новый элемент в список подписантов титула продавца "акта в формате 552-го приказа".
