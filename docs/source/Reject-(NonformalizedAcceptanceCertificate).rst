Reject 
===========================================

Метод объекта
:doc:`NonformalizedAcceptanceCertificate <NonformalizedAcceptanceCertificate>`.

**Синтаксис**


Reject(<Comment>)

**Параметры**


-  <Comment> (строка, обязательный) - комментарий к отказу в подписании.

**Возвращаемое значение**


Отсутствует

**Описание**


Формирует отказ в подписании документа.
