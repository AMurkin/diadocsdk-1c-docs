Send
====

Метод объекта :doc:`TransformTemplateTask <TransformTemplateTask>`


**Синтаксис**

Send()


**Возвращаемое значение**

:doc:`DocumentPackage <DocumentPackage>`.


**Описание**

Метод превращает шаблоны документов в документы. Результатом выполнения операции является сформированный пакет документов.
