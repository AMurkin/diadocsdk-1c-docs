GetBuyerContent 
===========================

Метод объекта :doc:`XmlTorg12 <XmlTorg12>`.

**Синтаксис**


GetBuyerContent()

**Возвращаемое значение**


:doc:`Torg12BuyerContent <Torg12BuyerContent>`.

**Описание**


Возвращает содержание документа торг-12 в виде объектной модели (титул
покупателя).
