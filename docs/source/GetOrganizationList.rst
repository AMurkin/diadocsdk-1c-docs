GetOrganizationList
===================

Метод объекта :doc:`DiadocConnection <Connection>`.

**Синтаксис**


GetOrganizationList()

**Возвращаемое значение**


:doc:`коллекция <Collection>` объектов
:doc:`Organization <Organization>`.

**Описание**


Возвращает список организаций в Диадоке, к которым пользователь имеет
доступ.
