SaveExternalCodes
=================

Метод объекта :doc:`Torg12SellerContent <Torg12SellerContent>`

**Синтаксис**


SaveExternalCodes()

**Возвращаемое значение**


Отсутствует

**Описание**


Позволяет сохранить список внешних идентификаторов товаров накладной.
Может быть использована только покупателем.
