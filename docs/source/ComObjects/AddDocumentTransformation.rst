AddDocumentTransformation
=========================

Метод объекта :doc:`TransformTemplateTask <TransformTemplateTask>`


**Синтаксис**

AddDocumentTransformation(<Id>)


**Параметры**

-  <Id> (строка, необязательный) - идентификатор сущности шаблона, на основе которой будет создаваться документ


**Возвращаемое значение**

:doc:`DocumentTransformation <DocumentTransformation>`


**Описание**

Добавляет в задание на трансформацию шаблона в документ информацию о том, какой на основе какого шаблона тот будет сформирован
