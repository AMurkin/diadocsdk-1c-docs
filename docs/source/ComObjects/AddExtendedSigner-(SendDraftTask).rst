AddExtendedSigner
==================

Метод объекта :doc:`SendDraftTask <SendDraftTask>`


**Синтаксис**

AddExtendedSigner()


**Возвращаемое значение**

:doc:`ExtendedSigner <ExtendedSigner>`


**Описание**

Добавляет новый элемент в список подписантов исходящего титула продавца УПД с отложенной отправкой.
