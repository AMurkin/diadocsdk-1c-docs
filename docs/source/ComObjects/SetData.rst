SetData
=======

Метод объекта :doc:`Organization <Organization>`.

**Синтаксис**


SetData(<Key>, <Value>)

**Параметры**


-  <Key> (строка, обязательный) - уникальный ключ пары в хранилище

-  <Value> (строка, обязательный) - значение

**Возвращаемое значение**


Отсутствует

**Описание**


Добавляет пару "ключ-значение" в хранилище.
