SaveContent
======================

Метод объекта :doc:`Document <Document>`.

Синтаксис
---------

SaveContent(<FilePath>)

Параметры
---------

-  <FilePath> (строка, обязательный) - путь для сохранения документа

Возвращаемое значение
---------------------

Отсутствует

Описание
--------

Сохраняет документ в виде файла по указанному пути.
